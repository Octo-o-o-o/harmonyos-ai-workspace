## 14. 备份策略

本地资料库（`upstream-docs/openharmony-docs/`）已经包含 OpenHarmony 全量官方文档。当上述任何外链失效时，先尝试在本地搜索：

```bash
cd upstream-docs/openharmony-docs
grep -rn "你要找的关键词" zh-cn/application-dev/
```

或用 ripgrep（如果安装）：

```bash
rg "@State|@Prop|@Link" zh-cn/application-dev/quick-start/
```
